HTML Reference Cheatsheet

Basic HTML Tags

Tag Attributes Name Description
<p> Paragraph Define a Paragraph
<div> class , id Division Divides off a section of code
<a> target,href Anchor Used to link to other locations ,target="_blank" for new web
<ul> Unordered list Lists elements in bullet points
<ol> Ordered list Lists elements in numbers or some ordered way
<li> List Used to define every item in ordered or Unordered list
<span> Span To apply some CSS to a very small part of text
<em> Emphasize Makes text italicized
<strong> Strong Makes text bold
<br> Break Works like endl

Multimedia Tags

Tag Attributes Name Description
<img> src,alt Image Adds an image , self closing
<video> src,controls,loop,autoplay Video Adds a Video ,content will be diplayed if video not visible
<audio> controls,autoplay Audio Insert audio file
<source> src,type="audio/mp3" Source Adds source for audi,used inside<audio>
<embed> src Embed Can embed any media content
